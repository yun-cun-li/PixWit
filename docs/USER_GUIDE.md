## 截图功能

### 快捷键

*快捷键可在托盘菜单"快捷键设置"中自定义*

### 区域选择

1. 按 **F1** 冻结屏幕
2. 拖动选区，拖动边缘调整大小
3. **右键/Enter/Space** 确认，**ESC** 取消

### 编辑工具

截图后自动打开编辑窗口：箭头、画笔、矩形、圆形、直线、马赛克、文字



### 截图设置

**打开方式**：托盘右键 → 截图设置

| 设置项 | 说明 |
|--------|------|
| 遮罩透明度 | 调整未选中区域的暗度（0-100%） |
| UI 检测模式 | 自动框选窗口/控件 |

**UI 检测模式**：

| 模式 | 说明 |
|------|------|
| 无 | 不自动框选，性能最优 |
| 仅窗口 | 框选窗口边界，速度快（推荐） |
| 仅元素 | 框选 UI 控件，速度较慢 |
| 混合 | 窗口+元素同时检测，最慢但最精确 |

## 取色器功能

在截图模式下按 **C键** 获取鼠标位置的颜色和坐标，自动复制到剪贴板。

**格式**：`RGB(255, 128, 64) | XY: 1024, 768`

## 图片查看器

### 打开图片

- 主界面 → "打开图片/视频"
- 历史截图 → 双击图片

**支持格式**：PNG、JPEG、WebP、BMP、GIF、SVG、ICO

### 查看操作

| 操作 | 功能 |
|------|------|
| 滚轮 | 以鼠标为中心缩放 |
| Ctrl+滚轮 | 强制缩放（任何模式） |
| 中键点击 | 跳到 100% 原始尺寸 |
| 拖动 / Space+拖动 | 平移图片 |
| 双击 | 切换适应窗口/原始大小 |
| 0 / 1 | 适应窗口 / 原始尺寸 |
| F | 全屏切换 |
| ← → / PageUp PageDown | 上一张/下一张 |
| Ctrl+← → | 任何模式下切换图片 |

### 编辑工具

**裁剪**：矩形裁剪、椭圆裁剪

**标注**：画笔、箭头、直线、圆形、矩形、马赛克、文字、序号
- 滚轮调整粗细（选中工具时）
- 绘制后 3 秒内可滚轮调粗细或 ESC 取消

**变换**：左旋转、右旋转、透视矫正

**格式转换**：保存时选择目标格式（PNG、JPEG、WebP、BMP、SVG、ICO）

## 视频处理

### 打开视频

主界面 → "打开图片/视频" → 选择视频文件

**支持格式**：mp4、avi、mov、mkv、flv、wmv 等

### 视频裁剪

- Ctrl+滚轮：缩放时间轴
- 滚轮：滚动时间轴
- 拖动左右滑块设置起止点
- 点击"保存裁剪"导出

### 视频压缩

点击"压缩"按钮，选择压缩级别：高质量(CRF 23)、平衡(CRF 28)、高压缩(CRF 32)

## OCR 文字识别

### 使用方法

截图或打开图片后，点击"OCR"按钮，结果自动复制到剪贴板。

### OCR 服务

| 服务 | 特点 |
|------|------|
| 本地 OCR（默认） | 离线、免费、首次需下载模型(~18MB) |
| 百度 OCR（可选） | 云端、准确率更高、需 API 密钥 |

配置百度 OCR：托盘 → AI 服务配置 → 百度 OCR 配置

## AI 翻译功能

截图后点击"翻译"按钮，自动 OCR 识别并翻译成中文。

**前置条件**：需配置 DeepSeek 或通义千问服务（托盘 → AI 服务配置）

---

## AI 服务配置

**入口**：托盘右键 → AI 服务配置

### DeepSeek（推荐用于翻译）

- 获取 API Key：https://platform.deepseek.com

### 通义千问（翻译备选）

- 获取 API Key：https://dashscope.aliyuncs.com

### 百度 OCR（文字识别）

- 获取密钥：https://cloud.baidu.com/product/ocr
- 启用后替代本地 OCR



## 常见问题

**Q：截图快捷键无效？**
- 检查快捷键设置，可能被其他软件占用


**Q：OCR 无法下载模型？**
- 检查网络，或使用百度 OCR 云服务

**Q：翻译/OCR 报错？**
- 在 AI 服务配置中测试连接，检查 API Key 和余额

**Q：录屏文件很大？**
- 使用"压缩"功能

**Q：Windows 7 能用吗？**
- 不支持，仅支持 Windows 10/11
