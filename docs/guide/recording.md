# 🎬 录屏功能

PixWit 提供专业的录屏功能，支持全屏录制、区域录制、录制时标注和智能音频检测。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| **F3** | 启动区域录制 |
| **F7** | 暂停/继续录制 |
| **F8** | 停止录制 |

## 区域录制

### 快速开始

1. **按 F3 启动区域录制**
   - 进入选区模式

2. **拖动鼠标绘制录制区域**
   - 选择要录制的屏幕区域
   - 可调整选区大小和位置

3. **点击"开始录制"按钮**
   - 3秒倒计时
   - 倒计时结束后开始录制

4. **按 F8 停止录制**
   - 录制完成，弹出保存提示
   - 可选择打开/编辑/查看文件夹

### 录制控制

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F7** | 暂停/继续 | 暂停录制，再按继续 |
| **F8** | 停止录制 | 结束录制并保存 |
| **ESC** | 退出/停止 | 标注模式：退出标注<br>非标注模式：停止录制 |

::: tip 提示
录制过程中，托盘图标会显示录制状态，右键可暂停或停止
:::

## 录制时标注 🎨

在录制过程中可实时标注画面，标注会嵌入录制内容。

### 标注快捷键

| 快捷键 | 功能 |
|--------|------|
| **Alt+1** | 画线条 |
| **Alt+2** | 画箭头 |
| **Alt+3** | 画圆形 |
| **Alt+4** | 画矩形 |
| **Alt+0** | 退出标注模式 |

### 标注说明

**注意事项：**
- ✅ 标注会实时嵌入录制画面
- ⚠️ 标注期间鼠标被占用，无法进行其他操作
- 🔄 按 **Alt+0** 或点击工具栏"穿透"按钮恢复鼠标

**使用技巧：**
- 适合制作教程视频
- 可以圈出重点内容
- 画箭头指示操作步骤

## 全屏录制

### 操作步骤

1. **打开主界面**
   - 双击托盘图标
   - 或托盘菜单 → "打开主界面"

2. **点击"录屏"按钮**
   - 选择"全屏录制"

3. **选择屏幕**
   - 多屏用户：选择要录制的屏幕
   - 单屏用户：直接开始录制

4. **按 F8 停止录制**
   - 录制完成，保存视频

### 录制状态提示

**全屏模式特色：**
- 屏幕四周绘制闪烁边框提醒录制中
- 边框不会被录入视频
- 系统托盘显示录制时长

## 音频设置 🎤

### 智能音频检测

**首次录制：**
- 自动检测系统音频和麦克风状态
- 弹出音频设置向导
- 根据提示完成配置

**音频选项：**

| 模式 | 说明 | 适用场景 |
|-----|------|---------|
| 仅麦克风 | 录制解说 | 教程讲解、配音 |
| 仅系统声音 | 录制游戏/视频声音 | 游戏录制、演示 |
| 麦克风+系统 | 混音录制 | 带解说的游戏/软件演示 |
| 静音 | 不录制声音 | 无声演示 |

### 手动配置音频

1. **打开设置**
   - 托盘菜单 → "录制设置"

2. **选择音频源**
   - 麦克风：选择输入设备
   - 系统声音：安装虚拟音频线

3. **测试音频**
   - 录制一小段测试
   - 播放检查音量

::: tip 录制系统声音
建议使用"虚拟音频线"（如 VB-Audio Cable）录制系统声音。
详细设置请参考项目文档。
:::

## 录制设置

### 画质设置

- **帧率**：30/60 FPS
- **质量**：高/中/低
- **编码器**：H.264

::: tip 建议
- 一般使用：30 FPS，中质量
- 高清录制：60 FPS，高质量
- 快速分享：30 FPS，低质量
:::

### 其他设置

- **显示鼠标光标**：录制时是否显示鼠标
- **显示鼠标点击**：高亮显示点击位置
- **录制边框**：区域录制时是否显示边框

## 录制后处理

### 录制完成对话框

录制完成后会弹出提示窗口：

| 按钮 | 功能 |
|-----|------|
| **打开** | 使用默认播放器播放视频 |
| **编辑** | 在 PixWit 中编辑视频 |
| **打开文件夹** | 打开视频所在文件夹 |

::: tip 提示
对话框不会自动关闭，操作完成后手动关闭
:::

### 视频编辑

点击"编辑"按钮，可以：
- ✂️ 裁剪视频（头尾、中间段）
- 🗜️ 压缩视频（减小体积）
- 🔄 格式转换（MP4/AVI/WebM/GIF）

详见 [视频处理](/guide/video)

## 高级技巧

### 分段录制

1. 按 **F7** 暂停录制
2. 准备下一段内容
3. 再按 **F7** 继续录制
4. 最终生成一个完整视频

**适用场景：**
- 需要切换窗口准备内容
- 避免录入不必要的操作

### 实时标注技巧

- 标注前先规划好要标注的内容
- 标注完成后立即按 **Alt+0** 退出
- 避免长时间占用鼠标

### 优化录制体积

**方法一：降低帧率**
- 设置中选择 30 FPS（默认）
- 非游戏录制足够流畅

**方法二：录制后压缩**
- 使用内置压缩功能
- 可减小 50%-80% 体积
- 画质损失很小

## 常见问题

### Q：录屏无声音？

**可能原因：**
- 未配置音频源
- 系统音频未安装虚拟音频线

**解决方法：**
1. 托盘菜单 → "录制设置"
2. 检查音频设置
3. 录制系统声音需安装虚拟音频线
4. 参考音频设置文档

### Q：录屏文件很大？

**解决方法：**
1. 录制后使用"压缩"功能
2. 或在录制设置中降低帧率/质量
3. 推荐：录制完成后压缩

### Q：区域录制边框闪烁影响录制？

**解决方法：**
- 托盘菜单 → "录制设置"
- 取消勾选"显示录制边框"

### Q：录制卡顿？

**可能原因：**
- 电脑性能不足
- 录制区域过大
- 帧率设置过高

**解决方法：**
1. 降低录制帧率（30 FPS）
2. 缩小录制区域
3. 关闭其他占用资源的程序
4. 降低录制质量

### Q：录制的视频在哪里？

**默认保存位置：**
- `%USERPROFILE%\Videos\PixWit\`
- 或录制时自选位置

**快速打开：**
- 录制完成提示 → "打开文件夹"
