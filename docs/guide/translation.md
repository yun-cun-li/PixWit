# 🌐  图片翻译

PixWit 内置 AI 智能翻译功能，可以自动识别图片中的外语文字并翻译成中文。

## 快速开始

### 一键翻译

1. 按 **F1** 截取包含外语文字的图片
2. 点击工具栏"**翻译**"按钮（🌐 图标）
3. 自动识别文字并翻译成中文
4. 查看翻译结果窗口

::: tip 自动化流程
点击"翻译"按钮后，系统会自动完成 OCR 识别 → 图片翻译 → 显示结果的全部流程
:::

## 翻译流程

**翻译按钮会自动完成：**
1. 📷 **OCR 识别** - 提取图片中的文字
2. 🌐 **图片翻译** - 调用 AI 服务进行翻译
3. 📝 **展示结果** - 原文和译文对照显示

**预计时间：**
- 短文本（<100字）：约 1-1.5 秒
- 中等文本（100-300字）：约 2-3 秒
- 长文本（>500字）：约 3-5 秒

### 提升速度技巧

**技巧1：精确截取**
- 只截取需要翻译的文字区域
- 避免包含过多无关内容
- 减少 OCR 和翻译时间

**技巧2：使用本地 OCR**
- 如果已配置百度 OCR，OCR 步骤会更快
- 本地 OCR 速度约 1-2 秒
- 云端 OCR 需要网络传输

**技巧3：网络环境**
- 使用稳定的网络连接
- 避免高峰期使用
- 如果超时，可重试

## 常见问题

### Q：翻译功能需要什么？

**必需条件：**
1. 配置 DeepSeek 或通义千问服务
2. 在 AI 服务配置中设置 API Key
3. 有网络连接

**配置方法：**
- 系统托盘右键 → **AI 服务配置**
- 填写对应服务的 API Key
- 测试连接后保存

---

### Q：翻译速度慢怎么办？

**正常翻译时间：**
- 短文本：1-2 秒
- 中等文本：2-3 秒
- 长文本：3-5 秒

**如果超过这个时间：**
1. 检查网络连接
2. 减少翻译文本长度
3. 重试翻译
4. 更换翻译服务

---

### Q：翻译要收费吗？

**费用说明：**

**DeepSeek：**
- 新用户有免费额度
- 免费额度用完后按量计费
- 价格：约 0.001 元/1000 字符

**通义千问：**
- 新用户有免费额度
- 免费额度用完后按量计费

:::

---

### Q：翻译结果不准确？

**可能原因：**
1. OCR 识别错误
2. 原文语法复杂
3. 专业术语多

**解决方法：**
1. 确保截图清晰，提高 OCR 准确率
2. 可以编辑识别的原文后重新翻译
3. 切换到百度 OCR 提高识别率
4. 使用专业翻译工具处理专业文档

---

### Q：支持哪些语言？

**当前状态：**
- 主要支持：英文 → 中文
- 理论支持：任何语言 → 中文

**说明：**
- AI 模型支持多语言识别
- 自动检测源语言
- 自动翻译成中文

---

### Q：能翻译成其他语言吗？

**当前状态：**
- 默认翻译成中文
- 暂不支持自定义目标语言

**计划：**
- 未来版本可能添加目标语言选择
- 支持中文翻译成其他语言

---

### Q：翻译失败怎么办？

**可能原因：**
1. 未配置 AI 服务
2. API Key 错误
3. 网络问题
4. API 额度用尽

**解决方法：**
1. 检查 AI 服务配置
2. 测试连接是否正常
3. 检查网络连接
4. 查看 API 账户余额

---

### Q：隐私安全吗？

**数据处理：**
- ⚠️ 图片和文字会上传到 AI 服务商
- ⚠️ 需要联网
- ℹ️ 受相应 AI 服务商隐私政策保护

**建议：**
- 非敏感内容可以放心使用
- 敏感文档建议使用本地翻译工具
