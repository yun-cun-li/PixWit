# 🌐 图片翻译

PixWit 内置 AI 智能翻译功能，可以自动识别图片中的外语文字并翻译成中文。

## 使用方法

1. 按 **F1** 截取包含外语文字的图片
2. 点击工具栏"**翻译**"按钮（🌐 图标）
3. 自动 OCR 识别并翻译成中文
4. 查看翻译结果（原文和译文对照）

::: tip 自动流程
点击"翻译"按钮后，系统自动完成：OCR 识别 → AI 翻译 → 显示结果
:::

## 前置条件

需要配置 **DeepSeek** 或 **通义千问** 翻译服务：

1. 系统托盘右键 → **AI 服务配置**
2. 填写对应服务的 API Key
3. 测试连接后保存

**获取 API Key：**
- DeepSeek（推荐）：https://platform.deepseek.com
- 通义千问：https://dashscope.aliyuncs.com

## 翻译结果窗口

**窗口布局：**
- 左侧：原图预览（可隐藏）
- 上方：原文内容
- 下方：中文翻译

**显示选项：** 隐藏图片 / 隐藏原文 / 显示全部

::: tip 智能记忆
窗口会记住您的显示偏好设置
:::

## 常见问题

**Q：翻译速度慢？**
- 短文本 1-2 秒，中等文本 2-3 秒，长文本 3-5 秒
- 检查网络连接

**Q：翻译要收费吗？**
- DeepSeek 和通义千问都有免费额度
- 超出后按量计费，成本极低

**Q：翻译/OCR 报错？**
- 在 AI 服务配置中测试连接
- 检查 API Key 和账户余额
