#  📊提取表格

## 概述

PixWit 表格提取功能是一款强大的 AI 驱动的表格识别工具，能够自动识别图片中的表格内容，并将其转换为可编辑的 HTML 格式。该功能支持复杂表格结构，包括合并单元格、表头、表尾等，特别适合处理截图、扫描件中的表格数据。
![一键复制](./一键复制到EXCEL.webp)
## 主要特性

### 🎯 智能识别
- **自动表格检测**：精准识别各类表格结构
- **复杂表格支持**：处理合并单元格、多行表头、多列布局
- **多种格式兼容**：支持打印表格、截图、手写表格等

### 📊 格式保留
- **完整结构**：保留表格的行列结构和合并关系
- **样式美观**：自动应用合适的样式和颜色
- **Excel 兼容**：直接复制粘贴到 Excel，格式完美保留



### 🔧 灵活配置
- **多服务支持**：PixWit AI 云服务、百度表格识别 API


## 服务选择

### PixWit AI（推荐）
- **优点**：无需配置，开箱即用
- **费用**：完全免费
- **网络**：需要联网
- **配置**：在设置中选择 "PixWit AI" 即可

### 百度表格识别
- **优点**：高准确率，企业级服务
- **费用**：按量计费（具体价格请查看百度云）
- **网络**：需要联网
- **配置**：需要申请百度云 API Key 和 Secret Key

## 使用方法

### 图形界面使用

1. **启动 PixWit**

2. **截取表格图片**
   - 按 F1 键进入截图模式
   - 选择包含表格的区域
   - 确认截图

3. **提取表格**
   - 在工具栏点击"提取表格"按钮
   - 等待 AI 识别完成（通常 1-3 秒）
   - 查看识别结果

4. **复制结果**
   - 点击"复制表格"按钮
   - 粘贴到 Excel 或其他表格软件
