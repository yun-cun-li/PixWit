# 安装指南

本页面将指导您完成 PixWit 的安装和初始设置。

## 📥 下载

前往 [下载页面](/download) 获取最新版本。

推荐下载 **绿色便携版**，无需安装，解压即用。

## 💻 系统要求

- **操作系统**：Windows 10 / 11（不支持 Windows 7）
- **架构**：x64 (64位)
- **内存**：建议 4GB 以上
- **磁盘空间**：约 100MB

::: warning Windows 7 不支持
PixWit 基于 Qt 6 框架开发，仅支持 Windows 10 (1809+) 和 Windows 11。
:::

## 🚀 安装步骤

### 方式一：绿色便携版（推荐）

1. **下载压缩包**
   - 下载 `PixWit-Portable-vX.X.X.zip`

2. **解压文件**
   - 解压到任意文件夹
   - 推荐位置：`C:\Program Files\PixWit` 或 `D:\Software\PixWit`

3. **运行程序**
   - 双击运行 `运行PixWit.bat` 或 `PixWit.exe`
   - 程序会最小化到系统托盘

4. **完成！**
   - 在任务栏右下角找到 PixWit 托盘图标
   - 右键点击查看菜单
   - 按 **F1** 开始第一次截图

::: tip 提示
便携版的所有配置和历史记录保存在 `%APPDATA%\PixWit` 目录，可随时备份。
:::

### 方式二：安装程序版

1. **下载安装程序**
   - 下载 `PixWit-Setup-vX.X.X.exe`

2. **运行安装**
   - 右键选择"以管理员身份运行"
   - 按照安装向导提示完成安装

3. **启动程序**
   - 安装完成后自动启动
   - 或从开始菜单启动

::: warning 杀毒软件拦截
如遇 Windows Defender 或其他杀毒软件拦截：
- 点击"更多信息" → "仍要运行"
- 或将程序添加到白名单
:::

## ⚙️ 初始设置

### 1. 设置开机自启（可选）

- 右键点击系统托盘图标
- 勾选"开机自动启动"

### 2. 自定义快捷键（可选）

- 托盘菜单 → "快捷键设置"
- 修改默认快捷键（如果与其他软件冲突）

### 3. 配置音频设置（录屏需要）

首次录屏时会自动提示音频设置向导。

如需手动配置：
- 托盘菜单 → "录制设置"
- 选择音频输入源

## 🔧 卸载

### 便携版
直接删除程序文件夹即可。

如需清理配置：
- 删除 `%APPDATA%\PixWit` 文件夹

### 安装版
1. 控制面板 → 程序和功能
2. 找到 PixWit
3. 点击"卸载"

## ❓ 安装问题

### 双击无反应？

**可能原因：**
1. 被杀毒软件拦截
2. 系统版本不符（需 Windows 10+）
3. 缺少运行库

**解决方法：**
1. 检查杀毒软件是否拦截
2. 确认系统版本
3. 尝试以管理员身份运行
4. 安装 VC++ Redistributable

### 启动后崩溃？

**可能原因：**
1. 系统版本过低
2. 缺少必要的系统组件

**解决方法：**
1. 确认 Windows 10 版本为 1809 或更高
2. 更新系统到最新版本
3. 查看错误日志：`%APPDATA%\PixWit\logs\`

### 提示缺少 DLL 文件？

**解决方法：**
1. 下载安装 [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 重启电脑后再试

### 快捷键无效？

**可能原因：**
1. 快捷键被其他程序占用
2. 快捷键未启用

**解决方法：**
1. 托盘菜单 → "快捷键设置"
2. 检查快捷键是否启用
3. 更换为其他快捷键
