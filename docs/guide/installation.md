# 📥 安装指南

本页面将指导您完成 PixWit 的安装和初始设置。

## 系统要求

- **操作系统**：Windows 10 (1809+) / Windows 11
- **内存**：建议 4GB 及以上
- **磁盘空间**：200MB

::: warning Windows 7 不支持
PixWit 基于 Qt 6 开发，仅支持 Windows 10 及以上系统
:::

## 下载安装

### 安装版

1. 前往 [下载页面](../download) 获取最新安装包
2. 双击运行 `PixWit-Setup.exe`
3. 按向导完成安装
4. 启动程序，托盘图标出现即表示运行成功

### 便携版

1. 下载便携版压缩包
2. 解压到任意目录
3. 运行 `PixWit.exe`
4. 无需安装，可直接使用

## 首次设置

### 基本配置

1. **快捷键**：托盘菜单 → "快捷键设置"，设置常用快捷键
2. **开机启动**：托盘菜单 → 勾选"开机自动启动"
3. **主题**：设置中心 → 选择深色/浅色主题

### AI 服务（可选）

使用 OCR、翻译、智能解读等功能需配置 AI 服务：

1. 托盘菜单 → "设置中心" → "🤖 AI 服务配置"
2. 选择服务来源：
   - **PixWit AI**（推荐）：无需配置，开箱即用
   - **自定义服务**：配置自己的 API Key
   - **本地模式**：仅使用本地 OCR

## 常见问题

### 启动失败

- 确认系统为 Windows 10 (1809+) 或更高
- 以管理员身份运行
- 安装 [VC++ 运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### 被杀毒软件拦截

PixWit 为开源软件，安全可靠：
- 点击"更多信息" → "仍要运行"
- 或将 PixWit 添加到白名单

## 卸载

### 安装版
控制面板 → 程序和功能 → 卸载 PixWit

### 便携版
直接删除文件夹即可

### 清理配置
删除 `%APPDATA%\PixWit` 目录可清理所有配置文件
