# 🔍 OCR 文字识别

快速识别图片中的文字并复制到剪贴板。

![PixWit OCR 文字识别功能界面 - 展示图片文字识别结果](./文字识别.webp)

## 快速开始

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+O** | 截图编辑窗口中触发 OCR |

## 使用方法

### 截图后识别

1. 按 **F1** 截图，选择包含文字的区域
2. 在编辑窗口按 **Ctrl+O** 或点击"OCR"按钮
3. 等待识别完成，结果自动显示
4. 点击"复制"或直接编辑文字

::: tip 快速复制
识别结果可直接复制，无需手动选中
:::

## OCR 服务

### PixWit AI（推荐）
- ✅ 无需配置，开箱即用
- ✅ 云端识别，准确率高
- ✅ 完全免费
- ⚠️ 需要联网

### 本地 OCR
- ✅ 完全离线，无需联网
- ✅ 隐私安全，数据不上传
- ✅ 识别速度快（1-2秒）
- ⚠️ 首次使用需下载模型（~18MB）

### 百度 OCR（可选）
- ✅ 企业级准确率
- ✅ 支持多种语言
- ⚠️ 需要 API Key
- ⚠️ 按量计费

## 服务配置

设置中心 → 🤖 AI 服务配置：
1. 选择服务来源（PixWit AI / 自定义 / 本地）
2. 如使用百度OCR，填入 API Key 和 Secret Key
3. 点击"测试"验证配置

## 提高识别率

### 截图技巧
- 放大后截图，确保文字清晰
- 只截取文字区域，避免过多背景
- 使用高对比度主题（黑字白底最佳）

### 支持的文字
- ✅ 简体中文
- ✅ 英文、数字
- ⚠️ 繁体中文（识别率较低）
- ⚠️ 手写字体（识别率较低）

## 常见问题

### Q：识别不准确？
- 确保图片清晰，文字足够大
- 提高对比度
- 避免艺术字体和手写字

### Q：本地模型下载失败？
1. 检查网络连接
2. 手动下载模型文件
3. 解压到 `PixWit安装目录\models\`
4. 重启 PixWit
