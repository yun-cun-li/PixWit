# 🔍 OCR 文字识别

PixWit 内置 OCR 文字识别功能，可以快速识别图片中的文字并复制到剪贴板。

## 使用方法

### 方法1：截图后识别

1. 按 **F1** 截图
2. 点击工具栏"**OCR**"按钮
3. 等待识别完成
4. 结果自动复制到剪贴板

### 方法2：打开图片识别

1. 主界面 → "打开图片/视频"
2. 选择图片文件
3. 点击"**OCR**"按钮

## OCR 服务选择

| 服务 | 特点 |
|------|------|
| 本地 OCR（默认） | 离线、免费、首次需下载模型(~18MB) |
| 百度 OCR（可选） | 云端、准确率更高、需 API 密钥 |

### 本地 OCR

- ✅ 完全离线，无需联网
- ✅ 隐私安全，数据不上传
- ✅ 识别速度快（约1-2秒）

**首次使用**：点击 OCR 按钮时会自动提示下载模型

### 百度 OCR（可选）

- ✅ 无需下载模型
- ✅ 识别准确率更高
- ✅ 支持手写、表格等

**配置方法**：托盘右键 → AI 服务配置 → 百度 OCR 配置

**获取密钥**：https://cloud.baidu.com/product/ocr

## 识别结果

**结果窗口：**
- 左侧：原图预览（可隐藏）
- 右侧：识别的文字（已自动复制）

**操作：** 可编辑文字、保存为 TXT

## 识别能力

| 内容 | 本地 OCR | 百度 OCR |
|------|----------|----------|
| 中文简体 | ✅ 优秀 | ✅ 优秀 |
| 英文/数字 | ✅ 优秀 | ✅ 优秀 |
| 手写字 | ⚠️ 一般 | ✅ 良好 |
| 表格 | ❌ 不支持 | ✅ 支持 |

::: tip 建议
重要文档用百度 OCR（更准），普通文档用本地 OCR（更快更安全）
:::

## 常见问题

**Q：OCR 无法下载模型？**
- 检查网络，或使用百度 OCR 云服务

**Q：识别不准确？**
- 确保图片清晰、文字够大
- 建议使用百度 OCR

**Q：如何切换 OCR 服务？**
- 托盘右键 → AI 服务配置 → 勾选/取消百度 OCR
