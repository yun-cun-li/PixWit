# 🔍 OCR 文字识别

PixWit 内置 OCR 文字识别功能，可以快速识别图片中的文字并复制到剪贴板。

## 使用方法

### 方法1：截图后识别

**快速流程：**
1. 按 **F1** 截图
2. 选择包含文字的区域
3. 点击工具栏"**OCR**"按钮
4. 等待识别完成
5. 结果自动复制到剪贴板

::: tip 提示
这是最常用的方式，可以快速识别屏幕上的任何文字
:::

### 方法2：打开图片识别

**操作步骤：**
1. 主界面点击"打开图片/视频"
2. 选择包含文字的图片文件
3. 点击工具栏"**OCR**"按钮
4. 等待识别完成
5. 查看识别结果

**适用场景：**
- 识别已保存的图片
- 批量识别多张图片
- 识别高分辨率图片

## 首次使用

### 自动下载模型

**首次使用流程：**
1. 点击"OCR"按钮
2. 弹出模型下载提示
3. 点击"下载"按钮
4. 等待下载完成（约 25MB）
5. 下载完成后自动开始识别

**下载说明：**
- 模型托管在 Gitee，国内访问快
- 下载窗口非模态，可继续使用其他功能
- 下载完成后系统通知
- 模型保存在应用目录 `models/` 文件夹

::: info 提示
模型只需下载一次，以后直接使用
:::

### 手动下载模型

如果自动下载失败，可以手动下载：

1. **下载模型文件**
   - 前往项目页面下载模型
   - 或从群文件下载

2. **放置模型文件**
   - 解压到 PixWit 安装目录的 `models/` 文件夹
   - 确保文件结构正确

3. **重启 PixWit**
   - 重新打开程序
   - OCR 功能即可使用

## 识别结果

### 结果窗口

识别完成后，弹出结果窗口：

**窗口布局：**
- **左侧**：原图预览
- **右侧**：识别的文字

**功能按钮：**
| 按钮 | 功能 |
|-----|------|
| 复制 | 复制文字到剪贴板（已自动复制）|
| 保存 | 导出为 TXT 文件 |
| 编辑 | 可直接修改识别文字 |
| 关闭 | 关闭结果窗口 |

::: tip 自动复制
识别结果已自动复制到剪贴板，可以直接粘贴使用
:::

### 结果编辑

**可以在结果窗口中：**
- 直接修改识别错误的文字
- 调整格式
- 添加或删除内容
- 保存为 TXT 文件

## 识别能力

### 支持的内容

| 内容类型 | 支持程度 | 说明 |
|---------|---------|------|
| ✅ 中文（简体）| 优秀 | 主要识别语言 |
| ✅ 英文 | 优秀 | 支持大小写 |
| ✅ 数字 | 优秀 | 0-9 |
| ✅ 多行文本 | 良好 | 自动保留换行 |
| ✅ 常用符号 | 良好 | 标点符号等 |
| ⚠️ 手写字 | 一般 | 识别准确率较低 |
| ⚠️ 繁体中文 | 一般 | 部分支持 |
| ❌ 竖排文字 | 不支持 | 仅支持横排 |

### 识别质量

**影响识别准确率的因素：**

1. **图片清晰度** ⭐⭐⭐
   - 清晰的图片识别率高
   - 模糊图片识别率低

2. **文字大小** ⭐⭐⭐
   - 文字太小难以识别
   - 建议文字高度 > 20 像素

3. **背景复杂度** ⭐⭐
   - 纯色背景识别率高
   - 复杂背景识别率低

4. **文字颜色** ⭐⭐
   - 黑字白底效果最好
   - 低对比度识别困难

5. **字体** ⭐
   - 规则字体识别率高
   - 艺术字体识别困难

## 使用技巧

### 提高识别准确率

**截图技巧：**
1. **精确选区**
   - 只截取包含文字的区域
   - 避免包含过多背景

2. **放大屏幕**
   - 如果文字太小，先放大网页/文档
   - 再截图识别

3. **调整对比度**
   - 使用高对比度主题
   - 或截图后调整亮度

**图片准备：**
1. **裁剪图片**
   - 去除多余部分
   - 只保留文字区域

2. **增强清晰度**
   - 提高图片分辨率
   - 降噪处理

3. **纠正倾斜**
   - 使用"矫正"功能
   - 确保文字水平

## 常见问题

### Q：OCR 无法下载模型？

**可能原因：**
- 网络连接问题
- 防火墙拦截

**解决方法：**
1. 检查网络连接
2. 关闭防火墙/代理
3. 重新点击"OCR"按钮
4. 或手动下载模型（见上方说明）

### Q：识别速度慢？

**可能原因：**
- 图片分辨率太高
- 电脑性能不足

**解决方法：**
1. 裁剪到文字区域
2. 降低图片分辨率
3. 关闭其他程序

**正常识别时间：**
- 小图（< 1MB）：1-3 秒
- 大图（> 5MB）：5-10 秒

### Q：识别不准确？

**可能原因：**
- 图片模糊
- 文字太小
- 背景复杂
- 手写字体

**解决方法：**
1. 提高图片质量
2. 放大后再截图
3. 使用纯色背景
4. 手写字改用印刷体

### Q：无法识别繁体中文？

**说明：**
- 当前模型主要针对简体中文
- 繁体中文识别率较低
- 可尝试识别，但准确率不保证

### Q：能识别表格吗？

**当前状态：**
- 可以识别表格中的文字
- 但不保留表格结构
- 结果为纯文本

### Q：识别结果能编辑吗？

**说明：**
- 可以在结果窗口中直接编辑
- 修改后可重新复制或保存
- 不会修改原图

### Q：能批量识别吗？

**当前状态：**
- 暂不支持批量识别
- 需要逐张处理

**建议：**
- 使用专业 OCR 软件（如 ABBYY）
- 或使用在线批量 OCR 服务
