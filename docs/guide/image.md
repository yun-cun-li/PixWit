# 🖼️ 图片处理

PixWit 不仅可以截图，还可以打开和编辑现有图片，支持多种格式转换。

## 打开图片

### 方法一：主界面打开

1. 双击托盘图标打开主界面
2. 点击"打开图片/视频"按钮
3. 选择图片文件
4. 在图片查看器中查看和编辑

### 方法二：拖拽打开

1. 直接拖拽图片文件到 PixWit 主窗口
2. 自动打开图片查看器

### 支持的格式

| 格式 | 说明 |
|-----|------|
| PNG | 无损压缩，支持透明 |
| JPEG/JPG | 适合照片，体积小 |
| BMP | 位图格式 |
| WebP | 现代格式，体积小质量高 |
| SVG | 矢量图（仅查看） |
| ICO | 图标格式 |

## 图片编辑工具

### 裁剪

**功能：**
- 选择保留区域
- 去除多余部分

**操作步骤：**
1. 点击"裁剪"按钮
2. 拖动选区选择保留区域
3. 调整选区大小和位置
4. 点击"确认"完成裁剪

### 旋转

**功能：**
- 顺时针旋转 90°
- 逆时针旋转 90°

**使用场景：**
- 纠正拍照方向
- 调整图片朝向

### 矫正

**功能：**
- 透视矫正
- 拍照文档纠偏

**使用场景：**
- 扫描歪斜的文档
- 拍照的书籍、白板等

**操作步骤：**
1. 点击"矫正"按钮
2. 拖动四个角点到文档四角
3. 点击"确认"完成矫正
4. 自动透视变换为矩形

### 标注

**可用工具：**
- 箭头 - 方向指示
- 画笔 - 自由绘画
- 矩形 - 框选重点
- 圆形 - 圈出区域
- 直线 - 画线
- 马赛克 - 打码
- 文字 - 添加说明

详见 [截图功能 - 编辑工具](./screenshot#编辑工具)

### OCR 识别

**功能：**
- 识别图片中的文字
- 自动复制到剪贴板

**操作：**
1. 点击"OCR"按钮
2. 等待识别完成
3. 查看结果

详见 [OCR 文字识别](./ocr)

## 格式转换

### 支持的目标格式

| 格式 | 特点 | 适用场景 |
|-----|------|---------|
| PNG | 无损压缩，支持透明 | 截图、图标、需要透明背景 |
| JPEG/JPG | 有损压缩，体积小 | 照片、分享图片 |
| WebP | 现代格式，质量高体积小 | 网页图片 |
| BMP | 无压缩，体积大 | 特殊需求 |
| ICO | 图标格式 | 应用图标、网站图标 |

### 转换方法

1. **打开图片**
   - 使用任何方法打开图片

2. **点击"保存"按钮**
   - 或按 **Ctrl+S**

3. **选择格式**
   - 在"保存类型"下拉框中选择目标格式

4. **设置选项**（部分格式）
   - JPEG：设置质量（1-100）
   - PNG：选择压缩级别

5. **保存**
   - 点击"保存"完成转换

::: tip 批量转换
目前不支持批量转换，每次转换一个文件。
:::

## 实用技巧

### 快速裁剪

1. 打开图片
2. 按 **C** 键快速进入裁剪模式
3. 调整选区
4. **Enter** 确认

### 快速格式转换

常见转换场景：

**PNG → JPEG（减小体积）：**
1. 打开 PNG
2. Ctrl+S 保存
3. 选择 JPEG 格式
4. 调整质量为 85

**JPEG → PNG（保留质量）：**
1. 打开 JPEG
2. Ctrl+S 保存
3. 选择 PNG 格式

**制作图标：**
1. 打开图片并裁剪为正方形
2. 保存为 ICO 格式
3. 选择尺寸（16/32/48/256）

### 文档纠偏

**适用场景：**
- 拍照的书籍、笔记
- 歪斜的扫描文档
- 白板拍照

**操作：**
1. 打开图片
2. 点击"矫正"
3. 四个角点对准文档四角
4. 确认完成

## 常见问题

### Q：支持哪些图片格式？

**输入格式：**
- PNG, JPEG, JPG, BMP, WebP, SVG, ICO

**输出格式：**
- PNG, JPEG, WebP, BMP, ICO

### Q：编辑后能撤销吗？

**撤销：**
- 按 **Ctrl+Z** 撤销
- 保存前可以多次撤销

**保存后：**
- 无法撤销，图片已永久修改
- 建议编辑前备份原图

### Q：如何批量处理图片？

**当前状态：**
- PixWit 暂不支持批量处理
- 需要逐个处理

**建议：**
- 使用专业批量处理工具
- 或使用脚本批量调用 PixWit

### Q：裁剪后能恢复吗？

**说明：**
- 裁剪并保存后无法恢复
- 建议操作前备份
- 或使用"另存为"保留原图

### Q：格式转换会损失质量吗？

**无损转换：**
- PNG ↔ BMP ↔ WebP（无损模式）

**有损转换：**
- 任何格式 → JPEG（有损）
- 转换时调整质量参数

### Q：为什么 SVG 不能编辑？

**说明：**
- SVG 是矢量格式
- PixWit 是位图编辑器
- SVG 仅支持查看和转换为位图
