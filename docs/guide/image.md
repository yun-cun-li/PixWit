# 🖼️ 图片处理

PixWit 不仅可以截图，还可以打开和编辑现有图片，支持多种格式转换。

## 打开图片

### 方法一：主界面打开

1. 双击托盘图标打开主界面
2. 点击"打开图片/视频"按钮
3. 选择图片文件
4. 在图片查看器中查看和编辑

### 方法二：拖拽打开

1. 直接拖拽图片文件到 PixWit 主窗口
2. 自动打开图片查看器

### 支持的格式

| 格式 | 说明 |
|-----|------|
| PNG | 无损压缩，支持透明 |
| JPEG/JPG | 适合照片，体积小 |
| BMP | 位图格式 |
| WebP | 现代格式，体积小质量高 |
| GIF | 动图格式 |
| SVG | 矢量图（仅查看） |
| ICO | 图标格式 |

## 图片查看器操作

| 操作 | 功能 |
|------|------|
| 滚轮 | 以鼠标为中心缩放 |
| Ctrl+滚轮 | 强制缩放（任何模式） |
| 中键点击 | 跳到 100% 原始尺寸 |
| 拖动 / Space+拖动 | 平移图片 |
| 双击 | 切换适应窗口/原始大小 |
| 0 / 1 | 适应窗口 / 原始尺寸 |
| F | 全屏切换 |
| ← → / PageUp PageDown | 上一张/下一张 |
| Ctrl+← → | 任何模式下切换图片 |

## 图片编辑工具

### 裁剪

**功能：**
- 选择保留区域
- 去除多余部分

**操作步骤：**
1. 点击"裁剪"按钮
2. 拖动选区选择保留区域
3. 调整选区大小和位置
4. 点击"确认"完成裁剪

### 旋转

**功能：**
- 顺时针旋转 90°
- 逆时针旋转 90°

**使用场景：**
- 纠正拍照方向
- 调整图片朝向

### 矫正

**功能：**
- 透视矫正
- 拍照文档纠偏

**使用场景：**
- 扫描歪斜的文档
- 拍照的书籍、白板等

**操作步骤：**
1. 点击"矫正"按钮
2. 拖动四个角点到文档四角
3. 点击"确认"完成矫正
4. 自动透视变换为矩形

### 标注

**可用工具：** 画笔、箭头、直线、圆形、矩形、马赛克、文字、序号

**操作技巧：**
- 滚轮调整粗细（选中工具时）
- 绘制后 3 秒内可滚轮调粗细或 ESC 取消

**变换操作：** 左旋转、右旋转、透视矫正


## 格式转换

### 支持的目标格式

| 格式 | 特点 | 适用场景 |
|-----|------|---------|
| PNG | 无损压缩，支持透明 | 截图、图标、需要透明背景 |
| JPEG/JPG | 有损压缩，体积小 | 照片、分享图片 |
| WebP | 现代格式，质量高体积小 | 网页图片 |
| BMP | 无压缩，体积大 | 特殊需求 |
| ICO | 图标格式 | 应用图标、网站图标 |

### 转换方法

1. **打开图片**
   - 使用任何方法打开图片

2. **点击"保存"按钮**
   - 或按 **Ctrl+S**

3. **选择格式**
   - 在"保存类型"下拉框中选择目标格式

4. **设置选项**（部分格式）
   - JPEG：设置质量（1-100）
   - PNG：选择压缩级别

5. **保存**
   - 点击"保存"完成转换
