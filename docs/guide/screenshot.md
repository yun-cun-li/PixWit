# 📸 截图功能

PixWit 提供强大而灵活的截图功能，支持区域截图、全屏截图、丰富的编辑工具和桌面贴图。

## 快速开始

### 基础快捷键

| 快捷键 | 功能 |
|--------|------|
| **F1** | 区域截图（冻结画面后拖动选择区域）|
| **F2** | 全屏截图（当前鼠标所在屏幕）|
| **Ctrl+Shift+A** | 截图后自动进入箭头标注模式 |

::: tip 自定义快捷键
具体快捷键可在托盘菜单"快捷键设置"中自定义
:::

## 区域截图

### 操作步骤

1. **按 F1 冻结屏幕**
   - 屏幕画面冻结，可以从容选择区域

2. **拖动鼠标绘制矩形选区**
   - 按住鼠标左键拖动
   - 会显示选区尺寸

3. **调整选区**
   - 拖动边缘或角点调整大小
   - 拖动选区内部移动位置
   - 8个调整手柄，精确控制

4. **确认截图**
   - **右键** / **Enter** / **Space**：确认截图，进入编辑
   - **ESC**：取消截图

::: tip 技巧
按住 **Shift** 拖动选区可等比缩放
:::

## 全屏截图

### 单屏截图

- 按 **F2** 截取当前鼠标所在屏幕
- 自动进入编辑窗口

### 多屏环境

- 鼠标移动到目标屏幕后按快捷键
- 会自动在鼠标所在屏幕截图
- 支持不同 DPI 的混合屏幕

## 编辑工具 🎨

截图后自动打开编辑窗口，提供以下工具：

### 工具列表

| 工具 | 快捷键 | 说明 |
|-----|--------|------|
| 🔺 箭头 | - | 方向指示，适合教程标注 |
| 🖌️ 画笔 | - | 自由绘画，手写批注 |
| ⬜ 矩形 | - | 绘制矩形框，框选重点 |
| ⭕ 圆形 | - | 绘制圆形/椭圆 |
| 📏 直线 | - | 绘制直线 |
| 🔳 马赛克 | - | 打码敏感信息 |
| 📝 文字 | - | 添加文字说明 |

### 工具设置

点击工具按钮后，可设置：
- **颜色**：多种预设颜色可选
- **粗细**：调整线条粗细
- **透明度**：部分工具支持

### 编辑操作

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+Z** | 撤销上一步操作 |
| **Ctrl+S** | 保存截图到文件 |
| **Ctrl+C** | 复制到剪贴板 |
| **ESC** | 关闭编辑窗口 |

::: tip 快速保存
双击编辑窗口外部区域可快速关闭窗口
:::

## 取色器功能 🎨

在截图模式下按 **C键** 可获取鼠标位置的颜色和坐标，自动复制到剪贴板。

**格式**：`RGB(255, 128, 64) | XY: 1024, 768`

## 截图设置 ⚙️

**打开方式**：托盘右键 → 截图设置

| 设置项 | 说明 |
|--------|------|
| 遮罩透明度 | 调整未选中区域的暗度（0-100%） |
| UI 检测模式 | 自动框选窗口/控件 |

**UI 检测模式**：

| 模式 | 说明 |
|------|------|
| 无 | 不自动框选，性能最优 |
| 仅窗口 | 框选窗口边界，速度快（推荐） |
| 仅元素 | 框选 UI 控件，速度较慢 |
| 混合 | 窗口+元素同时检测，最精确但最慢 |

## 特色功能

### 📌 贴图到桌面

1. 截图后，点击工具栏"贴图"按钮
2. 截图变为桌面便签，始终置顶
3. 可以拖动移动位置
4. 可以调整大小
5. 右键关闭贴图

**使用场景：**
- 临时参考资料
- 对照设计稿
- 记录重要信息

::: tip 设置
托盘菜单 → "隐藏贴图边框" 可隐藏边框，更美观
:::

### 🔍 OCR 文字识别

1. 截图后，点击工具栏"OCR"按钮
2. 等待识别完成
3. 自动弹出结果窗口
4. 文字已自动复制到剪贴板

详见 [OCR 文字识别](./ocr)

### 🌐 AI 智能翻译

1. 截图后，点击工具栏"翻译"按钮（🌐 图标）
2. 自动识别图片文字并翻译成中文
3. 弹出翻译结果窗口
4. 显示原文和译文对照

**前置条件：**
- 需要先配置 DeepSeek 或通义千问翻译服务
- 系统托盘右键 → AI 服务配置

详见 [ 图片翻译](./translation)

### 📜 历史记录

**自动保存：**
- 所有截图自动保存到历史文件夹
- 路径：`%APPDATA%\PixWit\ScreenshotHistory\images\`

**查看历史：**
- 托盘菜单 → "历史截图"
- 一键打开历史文件夹
- 按时间排序，方便查找

## 高级技巧

### 快速标注

使用组合快捷键直接进入标注模式：

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+Shift+A** | 箭头模式 |
| **Ctrl+Shift+R** | 矩形模式 |
| **Ctrl+Shift+T** | 文字模式 |

### 等比缩放

截图后按住 **Shift** 拖动选区角点，可保持宽高比缩放。

### 精确定位

- 使用方向键微调选区位置
- 按住 **Shift** + 方向键快速移动

## 常见问题

### Q：截图快捷键无效？

**解决方法：**
1. 检查托盘菜单 → 快捷键设置，确认已启用
2. 可能被其他软件占用，尝试更换快捷键
3. 以管理员身份运行 PixWit

### Q：多屏环境下如何截图？

**解决方法：**
- 鼠标移动到目标屏幕后按快捷键
- 会自动在鼠标所在屏幕截图
- 支持不同 DPI 的混合屏幕

### Q：截图保存在哪里？

**默认保存位置：**
- 历史截图：`%APPDATA%\PixWit\ScreenshotHistory\images\`
- 手动保存（Ctrl+S）：自选位置

### Q：如何设置默认保存格式？

**当前支持：**
- PNG（默认，无损）
- JPEG（适合照片）
- 保存时可选择格式

### Q：编辑后能再次编辑吗？

**说明：**
- 保存后的截图是静态图片
- 无法恢复标注前的状态
- 建议操作前先备份原图
