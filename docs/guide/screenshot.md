# 📸 截图功能

PixWit 提供强大的截图功能，支持区域截图、编辑标注和桌面贴图。

## 快速开始

| 快捷键 | 功能 |
|--------|------|
| **F1** | 区域截图（冻结画面后拖动选择区域）|

::: tip 自定义快捷键
所有快捷键可在托盘菜单"快捷键设置"中自定义
:::

## 区域截图

### 基本操作

1. **按 F1 冻结屏幕**
   - 屏幕画面冻结，从容选择区域

2. **拖动鼠标绘制选区**
   - 按住鼠标左键拖动
   - 实时显示选区尺寸

3. **调整选区**
   - 拖动边缘或角点调整大小
   - 拖动内部移动位置




## 编辑工具

### 标注工具

- 画笔：自由绘制
- 箭头：指示重点
- 直线：绘制直线
- 矩形：绘制矩形框
- 圆形：绘制圆形
- 马赛克：模糊敏感信息
- 文字：添加文字说明
- 序号：添加序号标记

### 编辑快捷键

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+Z** | 撤销 |
| **Ctrl+S** | 保存到文件 |
| **Ctrl+C** | 复制到剪贴板 |
| **ESC** | 关闭编辑窗口 |

## AI 功能

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+O** | OCR文字识别 |
| **Ctrl+E** | 提取表格 |
| **Ctrl+T** | 翻译 |
| **Ctrl+I** | 智能解读 |

## 截图设置

可在"设置中心 → 📸 截图设置"中配置：

### 显示设置
- **遮罩透明度**：调整未选中区域的暗度（0-100）
- **UI检测模式**：自动识别窗口/UI元素

### 功能选项
- **隐藏贴图边框**：贴图窗口无边框显示
- **大图自动缩放**：编辑窗口自动适应大图
- **拾色器**：启用取色工具
- **OCR索引**：历史截图OCR文本搜索

## 桌面贴图

### 使用方法

1. 截图后点击"贴图"按钮
2. 贴图窗口置顶显示
3. 可拖动、缩放
4. 右键关闭

### 应用场景
- 对照设计稿开发
- 参考教程操作
- 临时记录信息
