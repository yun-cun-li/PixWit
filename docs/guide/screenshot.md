# 📸 截图功能

PixWit 提供强大而灵活的截图功能，支持区域截图、全屏截图、丰富的编辑工具和桌面贴图。

## 快速开始

### 基础快捷键

| 快捷键 | 功能 |
|--------|------|
| **F1** | 区域截图（冻结画面后拖动选择区域）|


::: tip 自定义快捷键
具体快捷键可在托盘菜单"快捷键设置"中自定义
:::

## 区域截图

### 操作步骤

1. **按 F1 冻结屏幕**
   - 屏幕画面冻结，可以从容选择区域

2. **拖动鼠标绘制矩形选区**
   - 按住鼠标左键拖动
   - 会显示选区尺寸

3. **调整选区**
   - 拖动边缘或角点调整大小
   - 拖动选区内部移动位置
   - 8个调整手柄，精确控制

4. **确认截图**
   - **右键** / **Enter** / **Space**：确认截图，进入编辑
   - **ESC**：取消截图
