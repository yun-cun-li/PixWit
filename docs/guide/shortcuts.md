# ⌨️ 快捷键

PixWit 提供丰富的快捷键。所有快捷键可在"设置中心 → 快捷键设置"中自定义。

## 全局快捷键

这些快捷键在任何地方都可以使用。

| 快捷键 | 功能 |
|--------|------|
| **F1** | 区域截图 |
| **F3** | 启动区域录制 |
| **F6** | 暂停/继续录制 |
| **F7** | 停止录制 |

## 截图编辑

在截图编辑窗口中使用的快捷键。

### AI 功能

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+O** | OCR 文字识别 |
| **Ctrl+E** | 提取表格 |
| **Ctrl+T** | 翻译 |
| **Ctrl+I** | 智能解读 |

### 基础操作

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+Z** | 撤销 |
| **Ctrl+S** | 保存 |
| **Ctrl+C** | 复制 |
| **ESC** | 关闭窗口 |

## 录制标注

录制过程中的实时标注快捷键。

| 快捷键 | 功能 |
|--------|------|
| **Alt+I** | 线条工具 |
| **Alt+J** | 箭头工具 |
| **Alt+O** | 圆形工具 |
| **Alt+K** | 矩形工具 |
| **Alt+Esc** | 退出标注，恢复鼠标 |

::: warning 标注模式
标注期间鼠标被占用，无法进行其他操作。按 **Alt+Esc** 退出标注。
:::

## 视频编辑

视频编辑窗口中的快捷键。

| 快捷键 | 功能 |
|--------|------|
| **←/→** | 调整起始点（±0.1秒） |
| **↑/↓** | 调整结束点（±0.1秒） |
| **Ctrl+方向键** | 快速调整（±0.5秒） |
| **Ctrl+滚轮** | 缩放时间轴 |
| **滚轮** | 滚动时间轴 |
| **Enter** | 执行裁剪 |
| **ESC** | 关闭窗口 |

## 自定义快捷键

1. 右键托盘图标 → "设置中心"
2. 选择"⌨️ 快捷键设置"
3. 点击要修改的快捷键
4. 按下新的快捷键组合
5. 点击"应用"或"确定"保存

::: tip 避免冲突
如果快捷键与其他软件冲突，请更换为其他组合。
:::
