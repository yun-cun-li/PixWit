# ⌨️ 快捷键汇总

PixWit 提供丰富的快捷键，帮助您高效工作。所有快捷键都可以在"快捷键设置"中自定义。

## 全局快捷键

这些快捷键在任何地方都可以使用，无需聚焦 PixWit 窗口。

### 截图快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F1** | 区域截图 | 冻结屏幕后拖动选择区域 |
| **F2** | 全屏截图 | 截取当前鼠标所在屏幕 |
| **Ctrl+Shift+A** | 箭头模式截图 | 截图后自动进入箭头标注 |
| **Ctrl+Shift+R** | 矩形模式截图 | 截图后自动进入矩形标注 |
| **Ctrl+Shift+T** | 文字模式截图 | 截图后自动进入文字标注 |

### 录屏快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F3** | 启动区域录制 | 进入选区模式 |
| **F7** | 暂停/继续录制 | 暂停后可继续录制 |
| **F8** | 停止录制 | 结束录制并保存 |

::: tip 自定义快捷键
托盘菜单 → "快捷键设置" 可自定义所有全局快捷键
:::

## 录制标注快捷键

这些快捷键在录制过程中使用，用于实时标注。

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **Alt+1** | 画线条 | 进入线条标注模式 |
| **Alt+2** | 画箭头 | 进入箭头标注模式 |
| **Alt+3** | 画圆形 | 进入圆形标注模式 |
| **Alt+4** | 画矩形 | 进入矩形标注模式 |
| **Alt+0** | 退出标注 | 恢复鼠标控制 |
| **ESC** | 停止录制 | 仅在非标注模式下 |

::: warning 注意
标注期间鼠标被占用，无法进行其他操作。按 **Alt+0** 退出标注模式。
:::

## 自定义快捷键

### 如何自定义

1. **打开设置**
   - 右键托盘图标
   - 点击"快捷键设置"

2. **选择功能**
   - 在列表中选择要修改的功能
   - 点击当前快捷键

3. **设置新快捷键**
   - 按下新的快捷键组合
   - 点击"确定"保存

4. **冲突检测**
   - 如果快捷键已被占用
   - 系统会提示选择其他组合

::: warning 快捷键冲突
如果快捷键与其他软件冲突，可能无法正常使用。建议更换为不常用的组合。
:::

### 推荐配置

**方案1：Snipaste 风格**
- 截图：F1
- 贴图：F2
- 录屏：F3

**方案2：QQ 风格**
- 截图：Ctrl+Alt+A
- 录屏：Ctrl+Alt+R

**方案3：自定义**
- 根据个人习惯配置
- 避免与常用软件冲突

## 快捷键技巧

### 组合使用

**截图 + 标注：**
1. **Ctrl+Shift+A**：直接进入箭头模式
2. 截取区域
3. 立即标注，无需切换工具

**录屏 + 标注：**
1. **F3** 启动录制
2. **Alt+2** 进入箭头标注
3. 标注完成后 **Alt+0** 恢复鼠标
4. **F8** 停止录制

**视频编辑：**
1. **Ctrl+滚轮** 放大时间轴
2. **Space** 播放预览
3. **←/→** 逐帧定位
4. **I/O** 设置入出点
5. **Ctrl+S** 保存

### 效率提升

**快速截图流程：**
```
F1 → 选区 → Enter → Ctrl+C → ESC
```
（2秒完成截图复制）

**快速标注流程：**
```
Ctrl+Shift+A → 选区 → 画箭头 → Ctrl+S
```
（5秒完成箭头标注）

**快速录屏流程：**
```
F3 → 选区 → 开始 → [录制] → F8
```
（无需鼠标操作）

## 常见问题

### Q：快捷键无效？

**可能原因：**
1. 被其他软件占用
2. 快捷键未启用
3. 程序未以管理员身份运行

**解决方法：**
1. 检查"快捷键设置"中是否启用
2. 更换为其他快捷键
3. 以管理员身份运行 PixWit

### Q：如何查看所有快捷键？

**方法：**
1. 托盘菜单 → "快捷键设置"
2. 查看所有可用快捷键
3. 或查看本页面

### Q：能导入/导出快捷键配置吗？

**当前状态：**
- 暂不支持导入导出
- 配置保存在本地
- 重装后需要重新设置

### Q：快捷键在游戏中有效吗？

**说明：**
- 全屏游戏中可能无效
- 窗口模式或无边框模式下有效
- 部分游戏会屏蔽全局快捷键

### Q：能禁用某个快捷键吗？

**方法：**
1. 打开"快捷键设置"
2. 取消勾选对应功能
3. 该快捷键将被禁用
