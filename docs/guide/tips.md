# 💡 使用技巧

掌握这些技巧，让您的 PixWit 使用体验更上一层楼！

## 截图技巧

### 快速标注

**使用组合快捷键直接进入标注模式：**

| 快捷键 | 效果 |
|--------|------|
| **Ctrl+Shift+A** | 截图后直接进入箭头模式 |
| **Ctrl+Shift+R** | 截图后直接进入矩形模式 |
| **Ctrl+Shift+T** | 截图后直接进入文字模式 |

**适用场景：**
- 需要快速标注教程
- 频繁使用某个标注工具
- 提高标注效率

### 等比缩放选区

**操作方法：**
- 截图后按住 **Shift** 拖动选区角点
- 自动保持宽高比

**适用场景：**
- 截取正方形区域
- 保持特定比例
- 调整选区大小时不变形

### 精确定位选区

**方法1：方向键微调**
- 使用 **←/→/↑/↓** 移动选区（1像素）
- 按住 **Shift** + 方向键快速移动（10像素）

**方法2：数值输入**
- 在编辑窗口输入精确的宽高值
- 适合需要固定尺寸的截图

### 快速关闭编辑窗口

**方法1：双击外部**
- 双击编辑窗口外部区域
- 快速关闭窗口

**方法2：ESC 键**
- 按 **ESC** 关闭
- 不保存直接退出

### 多屏截图

**技巧：**
- 鼠标移动到目标屏幕
- 再按截图快捷键
- 自动在鼠标所在屏幕截图

**适用场景：**
- 双屏/多屏办公
- 截取副屏内容
- 跨屏截图

### 历史记录妙用

**快速查找历史截图：**
1. 托盘菜单 → "历史截图"
2. 按修改时间排序
3. 快速找到最近的截图

**技巧：**
- 所有截图自动保存
- 可以事后查找
- 避免重复截图

## 录屏技巧

### 分段录制

**操作流程：**
1. **F3** 启动录制
2. 录制第一段内容
3. **F7** 暂停
4. 准备第二段内容（切换窗口、准备材料）
5. **F7** 继续录制
6. 重复 3-5 步
7. **F8** 停止，生成完整视频

**优点：**
- 无需后期剪辑
- 避免录入准备过程
- 提高录制效率

**适用场景：**
- 软件教程录制
- 演示文稿讲解
- 游戏操作说明

### 实时标注技巧

**标注流程：**
1. 录制中按 **Alt+2** 进入箭头模式
2. 快速画箭头标注重点
3. 立即按 **Alt+0** 退出标注
4. 恢复鼠标，继续演示

**技巧：**
- 标注前规划好位置
- 快速标注，不要占用太长时间
- 标注完立即退出

**适用场景：**
- 教程视频制作
- 圈出重点内容
- 指示操作位置

### 优化录制体积

**方法1：降低帧率**
- 设置中选择 30 FPS
- 非游戏录制足够流畅
- 体积减小约 50%

**方法2：缩小录制区域**
- 只录制必要的窗口
- 避免全屏录制
- 减小分辨率

**方法3：录制后压缩**
- 使用内置压缩功能
- 选择"平衡"模式
- 可减小 50%-80% 体积

### 录制前准备

**检查清单：**
- ✅ 关闭不必要的通知（微信、QQ 等）
- ✅ 关闭弹窗广告
- ✅ 准备好演示内容
- ✅ 测试音频设置
- ✅ 清理桌面（全屏录制）

**技巧：**
- 使用"勿扰模式"
- 关闭自动更新
- 准备演示脚本

### 音频录制技巧

**仅麦克风模式：**
- 适合：讲解、配音
- 建议：使用外置麦克风
- 注意：保持适当距离

**仅系统声音模式：**
- 适合：游戏、播放视频
- 需要：安装虚拟音频线
- 注意：调整音量

**混音模式：**
- 适合：带解说的演示
- 注意：平衡两个音量
- 建议：先录制测试

## 视频编辑技巧

### 精确定位

**方法1：缩放时间轴**
1. **Ctrl+滚轮** 放大时间轴到 500%+
2. 可以看到每一帧
3. 精确到帧的裁剪

**方法2：逐帧播放**
1. 暂停播放
2. **←/→** 键逐帧前进/后退
3. 精确定位剪切点

**适用场景：**
- 需要精确剪辑
- 去除多余的帧
- 卡点剪辑

### 快速预览

**技巧：**
- 添加删除段后，双击列表项
- 右侧预览窗口自动播放该段
- 快速核对是否需要删除

**工作流程：**
1. 观看视频，记录需要删除的时间点
2. 逐个添加删除段
3. 双击每个删除段预览
4. 确认无误后导出

### 批量删除片段

**适用场景：**
- 去除视频中的多个错误
- 删除多个停顿
- 拼接多个有效片段

**技巧：**
- 从前往后添加删除段
- 使用列表查看所有删除段
- 删除段自动按时间排序

### GIF 制作技巧

**高质量 GIF 制作：**
1. **选择精彩片段**
   - 时长 3-5 秒
   - 包含完整动作

2. **设置参数**
   - 帧率：10-15 FPS
   - 分辨率：不超过 800px
   - 质量：高

3. **优化体积**
   - 去除音频
   - 裁剪到最小范围
   - 减少帧率

**适用场景：**
- 社交媒体分享
- 论坛发帖
- 动态表情包

### 视频压缩策略

**场景1：高质量保存**
- 选择"高质量"（CRF 23）
- 适合：存档、备份
- 体积：减小 30%-50%

**场景2：一般分享**
- 选择"平衡"（CRF 28）
- 适合：微信、QQ 分享
- 体积：减小 50%-70%

**场景3：快速分享**
- 选择"高压缩"（CRF 32）
- 适合：临时分享、预览
- 体积：减小 70%-80%

## OCR 技巧

### 提高识别准确率

**技巧1：放大后截图**
1. 如果文字太小，先放大网页/文档
2. 放大到 150%-200%
3. 再截图识别

**技巧2：精确选区**
- 只截取包含文字的区域
- 避免包含过多背景
- 提高识别速度和准确率

**技巧3：调整对比度**
- 使用高对比度主题
- 黑字白底效果最好
- 避免低对比度

### 快速识别流程

**流程：**
```
F1 → 选择文字区域 → OCR → 自动复制 → 粘贴使用
```

**时间：** 5秒完成

**适用场景：**
- 快速复制网页文字
- 提取图片中的文字
- 识别 PDF 中的文字

### 识别长文本

**技巧：**
1. 分段截取
2. 逐段识别
3. 合并结果

**或者：**
1. 截取整个页面
2. OCR 识别
3. 人工校对

### 表格识别技巧

**当前限制：**
- OCR 不保留表格结构
- 结果为纯文本

**变通方法：**
1. 逐行截取识别
2. 手动整理为表格
3. 或使用专业 OCR 软件

## 效率技巧

### 键盘流操作

**截图复制（全键盘）：**
```
F1 → 拖动鼠标 → Enter → Ctrl+C → ESC
```

**快速标注（全键盘）：**
```
Ctrl+Shift+A → 拖动鼠标 → Enter → 标注 → Ctrl+S
```

**录屏（全键盘）：**
```
F3 → 拖动鼠标 → Enter → [录制] → F8
```

### 工作流程优化

**教程制作流程：**
1. F3 启动录制
2. Alt+2 标注重点
3. Alt+0 恢复鼠标
4. 继续演示
5. F8 停止
6. "编辑" → 删除中间段
7. "压缩" → 保存

**截图标注流程：**
1. Ctrl+Shift+A 箭头模式截图
2. 快速标注
3. Ctrl+S 保存
4. 或 Ctrl+C 复制

### 批量任务处理

**批量截图：**
1. 准备所有需要截图的内容
2. 逐个按 F1 截图
3. 所有截图自动保存到历史
4. 统一整理历史截图

**批量视频处理：**
1. 录制多个视频
2. 逐个打开编辑
3. 批量压缩
4. 统一保存到文件夹

## 常见场景技巧

### 制作软件教程

**推荐流程：**
1. 准备演示脚本
2. 关闭通知
3. F3 区域录制（录制软件窗口）
4. 录制时使用 Alt+2 标注重点
5. 使用 F7 暂停准备下一步
6. F8 停止
7. 删除中间段
8. 压缩导出

### 快速反馈 Bug

**推荐流程：**
1. F1 截取 Bug 界面
2. 使用箭头和文字标注
3. Ctrl+S 保存
4. 发送给开发人员

### 保存网页内容

**方法1：截图保存**
- F1 截取网页内容
- 自动保存到历史

**方法2：OCR 提取文字**
- F1 截取文字部分
- OCR 识别
- 保存为 TXT

### 制作动图

**推荐流程：**
1. F3 录制操作（3-5 秒）
2. F8 停止
3. "编辑" → 裁剪到精彩部分
4. "格式转换" → GIF
5. 设置参数（帧率 10，分辨率 640）
6. 导出

## 高级技巧

### 自动化工作流

**设置开机自启：**
- 托盘菜单 → "开机自动启动"
- 开机后自动运行
- 随时可用

**固定快捷键：**
- 使用不冲突的快捷键
- 养成肌肉记忆
- 提高效率

### 配置备份

**定期备份：**
- 配置路径：`%APPDATA%\PixWit\`
- 包含：设置、快捷键、历史记录
- 重装前备份，重装后恢复

### 性能优化

**优化建议：**
1. 定期清理历史截图
2. 关闭不必要的特效
3. 降低录制质量（如果卡顿）
4. 关闭其他占用资源的程序
