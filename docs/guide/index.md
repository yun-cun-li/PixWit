# PixWit 使用手册

欢迎使用 PixWit！

**适用版本**：PixWit 1.1.x 及以上


## 🚀 快速入门

### 首次使用

1. **启动程序**：运行 `PixWit.exe`，程序会最小化到系统托盘
2. **查看托盘图标**：在任务栏右下角找到 PixWit 图标
3. **右键菜单**：右键点击托盘图标，查看所有功能
4. **开始使用**：按 **F1** 开始第一次截图！

### 核心快捷键

| 快捷键 | 功能 |
|--------|------|
| **F1** | 区域截图 |
| **F3** | 启动区域录制 |
| **F6** | 暂停/继续录制 |
| **F7** | 停止录制 |

*所有快捷键可在托盘菜单"快捷键设置"中自定义*
