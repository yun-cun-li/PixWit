
# 智能解读

PixWit 智能解读功能是一款基于大语言模型（LLM）的智能内容分析工具，能够深度理解和解读图片中的文本内容。该功能不仅能识别文字，还能进行语义分析、逻辑推理和实用建议，提供全方位的内容解读服务。
 ![智能解读](./解读.webp)

## 主要特性

### 🧠 深度智能分析
- **语义理解**：不仅识别文字，还能理解上下文和语义关系
- **多模态融合**：结合OCR文字识别与AI语义分析



### ⚡ 实时交互
- **流式输出**：实时显示分析结果，无需等待
- **即时复制**：一键复制解读结果

### 🔧 灵活配置
- **多服务支持**：PixWit AI云服务、DeepSeek、通义千问
- **性能优化**：响应速度优化

## 使用方法

### 图形界面使用

1. **启动 PixWit**

2. **截取分析对象**
   - 按 F1 键进入截图模式
   - 选择包含待分析内容的区域
   - 确认截图进入编辑界面

3. **触发智能解读**
   - 点击工具栏的"解读"按钮（💡图标）
   - 或使用快捷键 Ctrl+I
   - 等待AI分析完成

4. **查看解读结果**
   - 左侧显示原图，右侧显示解读内容
   - 点击"📷显示图片"可在左右视图间切换
   - 点击"复制解读"保存结果到剪贴板



### 快捷键操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl+I | 触发智能解读 | 在截图编辑窗口中生效 |
| F1 | 截图模式 | 开始新的截图 |
| ESC | 取消操作 | 在各种界面中通用 |
